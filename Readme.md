##  Instructions :- 

    1. Download or clone the project.
    2. In project directory perform -    npm i
    3. Now create a   ( .env )   file to store
       a. Database Username
       b. Database password
       c. Database Name
       d. Database Collection Name
       e. Database cluster name
       f. PORT
